API Fixes Required

1

enabled_metric block removed (azurerm 4.x)

The enabled_metric { category = "AllMetrics" } block in azurerm_monitor_diagnostic_setting was dropped in the azurerm 4.x provider. Metrics are captured implicitly — the block was removed.

2

NSG flow logs creation blocked since 2025-06-30

Azure retired new NSG flow log creation. Both fl-app-dev and fl-mgmt-dev were migrated from network_security_group_id to target_resource_id pointing to the App and Mgmt VNets (VNet flow logs).

3

AZFWDnsProxy log category unsupported

The AZFWDnsProxy diagnostic category is not available for Azure Firewall Standard tier in Sweden Central. Removed from the firewall diagnostic setting — other five categories (NetworkRule, AppRule, NatRule, ThreatIntel, IdpsSignature) are retained.

Key Code Change — Flow Log Migration

 resource "azurerm_network_watcher_flow_log" "app" {
-  network_security_group_id = azurerm_network_security_group.app.id
+  target_resource_id        = azurerm_virtual_network.app.id
-  version                   = 2
   storage_account_id        = azurerm_storage_account.flow_logs.id
   enabled                   = true
   ...
 }

 resource "azurerm_network_watcher_flow_log" "mgmt" {
-  network_security_group_id = azurerm_network_security_group.mgmt.id
+  target_resource_id        = azurerm_virtual_network.mgmt.id
-  version                   = 2
   ...
 }

 # firewall diagnostic setting
